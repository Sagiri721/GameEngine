Hello, World! Welcome to HimawariServer

HimawariServer is a service that creates and set's up for you a blazingly fast webserver with crow and C++ for you to create a remote database connection and send information between your game's clients with websockets'

Installing

Opened this file without the server set up? To get the server up and running you're going to need a C++ compiler, the default Visual Studio Community compiler should work fine. Using the GNU compiler is also a viable choice but might take a little more trobleshooting work

Install Visual Studio Community
Install the GNU compiler

After getting the C++ compiler you are going to need the Crow library header file, alternatively if you're not using Visual Studio you might want to download Crow from vcpkg, and add it to the same folder as the Main.cpp file

Install vcpkg
Get Crow

If everything went well you should be able to now run the web server by executing Main.cpp!

Have fun creating with Himawari2D!!
